## Getting  Started 
```bash
# Clone repo
git clone https://github.com/0xNonso/nft-perp-trader.git
cd nft-perp-trader

# Checkout branch
git checkout dev
```

## Test
Tests use [Foundry: Forge](https://github.com/gakonst/foundry)
```bash
# Install dependencies
forge install

# Run tests
forge test -vvvv
```
